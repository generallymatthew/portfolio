# Portfolio Site Research

This folder holds the UX research and strategy documents for matthewbartonux.com.

## Files

- [goals-and-outcomes.md](goals-and-outcomes.md) — Site goals, success metrics, and open questions
- [target-audience.md](target-audience.md) — Who visits the site and what they need
- [design-audit.md](design-audit.md) — Ongoing checklist: is the design meeting its goals?

## How to use this

1. Start with **goals-and-outcomes.md** to set or revisit the north star for the site.
2. Use **target-audience.md** to sense-check copy and layout decisions against real visitor needs.
3. Run through **design-audit.md** after any significant update to catch regressions and gaps.
